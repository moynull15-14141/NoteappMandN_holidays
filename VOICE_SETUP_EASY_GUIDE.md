# Windows 10/11 এ Noteapp Voice Recording Setup - সহজ পদক্ষেপ

## ১ম ধাপ: Windows Settings খুলুন
```
Windows Key + I
```
অথবা Start > Settings

## ২য় ধাপ: Microphone Settings এ যান
```
Settings 
→ Privacy & Security (বাম পাশ)
→ Microphone (স্ক্রল করে খুঁজুন)
```

## ৩য় ধাপ: Desktop Apps এর জন্য মাইক Enable করুন

আপনি এই সেটিংস দেখবেন:
- `Allow apps to access your microphone` - এটি **On** থাকা উচিত
- `Allow desktop apps to access your microphone` - এটিও **On** করুন

## ৪র্থ ধাপ: Noteapp Add করুন

**যদি Noteapp এখনো লিস্টে না থাকে:**
1. নিচে স্ক্রল করুন
2. প্রথমবার Noteapp চালিয়ে মাইক ব্যবহার করুন
3. একটি dialog আসবে - **Allow** করুন
4. তারপর Settings এ দেখা যাবে

**Noteapp কে On/Off করার জন্য:**
1. Microphone settings এ নিচে স্ক্রল করুন
2. Noteapp খুঁজুন
3. Toggle button **On** করুন (সবুজ হয়ে যাবে)

## ৫ম ধাপ: Speech Language ইনস্টল করুন (গুরুত্বপূর্ণ!)

```
Settings 
→ Time & Language
→ Speech
```

এখানে:
1. **Speech language** চেক করুন
2. যদি **English (United States)** না থাকে:
   - এর পাশে **ডাউনলোড** বাটন ক্লিক করুন
   - ডাউনলোড সম্পন্ন হওয়ার অপেক্ষা করুন (কয়েক মিনিট লাগতে পারে)

## ৬ষ্ঠ ধাপ: Noteapp এ Voice Recording ব্যবহার করুন

1. **Noteapp খুলুন**
2. **নতুন এন্ট্রি** বোতাম ক্লিক করুন
3. **"Write your thoughts..."** এর ডান দিকে **মাইক আইকন** দেখা যাবে
4. **মাইক আইকন ক্লিক করুন** (লাল হয়ে উঠবে)
5. **স্পষ্টভাবে কথা বলুন**
6. **আবার মাইক আইকন ক্লিক করুন** রেকর্ডিং বন্ধ করতে
7. আপনার কথা **টেক্সটে পরিণত হবে**!

## সাধারণ সমস্যা এবং সমাধান

| সমস্যা | সমাধান |
|--------|--------|
| Dialog box আসছে না | প্রথমবার নিজেই সেটিংসে ম্যানুয়ালি Allow করুন |
| Voice recognize হচ্ছে না | Speech Language ইনস্টল আছে কিনা চেক করুন |
| কোনো সাউন্ড নেই | Control Panel > Sound > Recording এ মাইক চেক করুন |
| পটভূমিতে অনেক শব্দ | শান্ত জায়গায় চেষ্টা করুন |

- [ ] মাইক physically connected
- [ ] Control Panel এ মাইক দেখা যাচ্ছে
